GitHub Pages 部署步驟

  • 在 GitHub 建立 Repository 並命名。
  • 新增檔案 index.html 並貼入本頁代碼。
  • 進入 Settings > Pages，將 Branch 設為 main 並儲存。
  • 等待 1 分鐘後，存取 GitHub 提供給您的網址。
  • 最重要： 將 GitHub 網址複製並更新到 LINE 的 Endpoint URL。
