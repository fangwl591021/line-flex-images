LINE Flex Message Editor (GitHub Pages Version)

這是一個專為 LINE Flex Message 設計的視覺化編輯器，特別優化了「電商商品列表」的製作流程。

主要功能

Hero 視覺：支援圖片、影片或隱藏，可自訂比例。

Body 商品：支援 3 欄或 4 欄佈局，可切換純色背景或圖片背景（支援等比縮放模式）。

Footer 按鈕：可自訂背景色、說明文字（支援換行、對齊、變色）與雙按鈕。

即時預覽：右側提供模擬 LINE 聊天室的預覽畫面。

一鍵輸出：點擊按鈕即可複製標準 Flex Message JSON。

如何在 GitHub 上使用

您不需要任何伺服器，只需利用 GitHub Pages 即可免費託管此工具。

建立 Repository：在 GitHub 上建立一個新的公開 Repository（例如 line-flex-editor）。

上傳檔案：將 index.html 上傳到該 Repository 的根目錄。

開啟 Pages：

進入 Repository 的 Settings > Pages。

在 Build and deployment > Branch 選擇 main (或 master) > / (root)。

點擊 Save。

取得網址：等待幾分鐘後，GitHub 會提供一個網址（例如 https://yourname.github.io/line-flex-editor/），打開即可使用！

本地使用

直接用瀏覽器（Chrome / Edge / Safari）打開 index.html 亦可直接使用，無需安裝任何軟體。
